portly

A lightweight Windows system tray app that lists active TCP ports in real time and lets you terminate processes with a single click when needed.

Windows 10 / 11 Tauri 2.x MIT License

1. Installation

To start using portly, follow these steps:

  1. Go to the GitHub Releases page:
    Releases — portly v0.1.0
  2. Download the portly_0.1.0_x64-setup.zip file and extract it.
  3. Run the installer, approve the UAC prompt, and complete the setup.
Run as Administrator to see all system ports.

2. Main Screen

When you open the app, active TCP ports are listed with process name and directory info. The footer shows the live port count.

3. Search

Use the search bar at the top to instantly filter the list by port number, process name, or directory path.

  • Port number: :3000 or 3000
  • Process name: node, nginx
  • Directory path: your project folder name

4. Quick Actions

Each port row has three quick actions:

  • >_ Open terminal — Opens a terminal in the process directory.
  • 📁 Open folder — Shows the directory in Explorer.
  • ✕ Kill process — Terminates the process after confirmation.

5. Settings

Click the Options button in the bottom-left to change the refresh interval. The list updates automatically at your chosen interval.

  • 3 seconds — busy development environment
  • 5 seconds — default
  • 10 / 30 seconds — lower resource usage

6. System Tray

portly runs in the background from the system tray without cluttering the taskbar. Left-click the tray icon to show or hide the window.

If the tray icon is missing, check hidden icons in Windows system tray settings.
